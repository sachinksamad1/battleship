# Security Policy

## Supported Versions

Currently, the `main` branch is the only supported version.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please DO NOT open a public issue. 

Instead, please send an email to the project maintainers directly or use GitHub's private vulnerability reporting feature (if enabled for the repository). We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.

We take all security issues seriously and will respond to reports as quickly as possible.
