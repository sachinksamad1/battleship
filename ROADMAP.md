# Battleship Project Roadmap

This document outlines the planned future enhancements for the Battleship project.

## Phase 4: Frontend Integration & UI Polish (Current Focus)
- [ ] **Animations:** Implement explosion and splash CSS animations for hits and misses.
- [ ] **Audio:** Integrate `howler.js` for placement thuds, cannon fire, hits, misses, and victory/defeat themes.
- [ ] **Transitions:** Add smooth slide/fade transitions when swapping turns or phases.
- [ ] **Settings Panel:** Provide a UI for users to toggle sound and visual effects.

## Phase 5: Future Enhancements (Planned)
- [ ] **Spectator Mode:** Allow users to join a live room as a spectator.
- [ ] **Replay System:** Save game logs and allow users to watch a playback of previous matches.
- [ ] **Ranked Matchmaking:** Introduce a backend database to track ELO ratings and match players automatically.
- [ ] **Chat System:** Basic emoji/text chat within multiplayer rooms.
